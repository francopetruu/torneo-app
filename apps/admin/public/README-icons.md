# PWA Icons

This directory should contain the following PWA icons:

## Required Icons

- `pwa-192x192.png` - 192x192 pixels, PNG format
- `pwa-512x512.png` - 512x512 pixels, PNG format
- `favicon.ico` - Standard favicon
- `apple-touch-icon.png` - 180x180 pixels for iOS
- `mask-icon.svg` - SVG icon for Safari pinned tabs

## Icon Design Guidelines

- Use a simple, recognizable design that works at small sizes
- Ensure good contrast for visibility
- Follow your app's brand colors
- Icons should be square with some padding (not edge-to-edge)

## Generating Icons

You can use online tools like:

- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Favicon.io](https://favicon.io/)

Or create them manually using design tools like Figma, Adobe Illustrator, or Canva.

## Placeholder Icons

For development, you can create simple placeholder icons:

- Solid color squares with text/logo
- Use your app's primary color (#1e40af for admin)
- Add "A" for Admin or your logo
